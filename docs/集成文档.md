# vPointer 对接文档

## 概述

本文档描述了如何在Windows C++程序中对接vPointer Android APP，实现虚拟光标的远程控制。

## vPointer APP 通信协议

### 协议概述

| 属性 | 值 |
|------|-----|
| 传输协议 | UDP |
| 默认端口 | 6533 |
| 数据格式 | 文本（逗号分隔） |

### 发送数据格式（Windows → Android）

数据格式为5个整数值，用逗号分隔：

```
x,y,show,down,orientation
```

| 字段 | 类型 | 说明 |
|------|------|------|
| x | int | 光标X坐标（像素） |
| y | int | 光标Y坐标（像素） |
| show | int | 1=显示光标, 0=隐藏光标 |
| down | int | 1=按下状态, 0=正常状态 |
| orientation | int | 设备方向（0-3） |

**示例：**
```
"100,200,1,0,0"   // 在(100,200)显示光标，未按下
"300,400,1,1,0"   // 在(300,400)显示光标，按下状态
"0,0,0,0,0"       // 隐藏光标
```

### 接收数据格式（Android → Windows）

单字节数据，表示设备方向：

| 值 | 方向 | 角度 |
|-----|------|------|
| 0x00 | 竖屏正常 | 0° |
| 0x01 | 横屏向右 | 90° |
| 0x02 | 竖屏倒置 | 180° |
| 0x03 | 横屏向左 | 270° |

## DLL 架构设计

### 设计理念

本DLL采用**适配器模式（Adapter Pattern）**设计，具有以下特点：

1. **高扩展性**：通过抽象接口支持多种虚拟指针APP
2. **低耦合**：业务逻辑与通信协议分离
3. **易维护**：新增APP只需实现适配器接口
4. **双接口**：同时提供C++和C接口

### 架构图

```
┌─────────────────────────────────────────────────────────────┐
│                    应用程序 (Your App)                       │
├─────────────────────────────────────────────────────────────┤
│  C++接口  │  C接口                                          │
│  IVirtualPointerAdapter │ VPTR_HANDLE                      │
├─────────────────────────────────────────────────────────────┤
│                    VirtualPointerDLL                         │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │           IVirtualPointerAdapter (抽象接口)          │   │
│  │  - Initialize()  - MovePointer()  - ShowPointer()   │   │
│  │  - Connect()     - HidePointer()  - SetPressed()    │   │
│  └─────────────────────────────────────────────────────┘   │
│                              │                               │
│              ┌───────────────┼───────────────┐               │
│              ▼               ▼               ▼               │
│  ┌─────────────────┐ ┌─────────────┐ ┌──────────────┐       │
│  │ VPointerAdapter │ │ OtherApp1   │ │ OtherApp2    │       │
│  │  (vPointer APP) │ │  Adapter    │ │  Adapter     │       │
│  └─────────────────┘ └─────────────┘ └──────────────┘       │
├─────────────────────────────────────────────────────────────┤
│                    网络通信层 (UDP)                          │
└─────────────────────────────────────────────────────────────┘
```

## 快速开始

### 1. 编译DLL

#### 使用CMake

```bash
# 创建构建目录
mkdir build && cd build

# 生成Visual Studio项目
cmake .. -G "Visual Studio 17 2022" -A x64

# 编译
cmake --build . --config Release
```

#### 使用Visual Studio

1. 打开Visual Studio
2. 选择"打开本地文件夹"，选择项目根目录
3. 选择Release/x64配置
4. 生成解决方案

### 2. 集成到项目

#### 方法一：动态链接

```cpp
// 1. 包含头文件
#include "VirtualPointerCAPI.h"

// 2. 加载DLL
HMODULE hDll = LoadLibraryA("VirtualPointer.dll");

// 3. 获取函数指针
typedef VPTR_HANDLE (*CreateAdapterFunc)(const char*);
CreateAdapterFunc createAdapter = (CreateAdapterFunc)GetProcAddress(hDll, "VPTR_CreateAdapter");

// 4. 创建适配器
VPTR_HANDLE handle = createAdapter("vPointer");
```

#### 方法二：静态链接（推荐）

```cpp
// 1. 包含头文件
#include "VirtualPointerCAPI.h"

// 2. 链接导入库
// 在项目设置中添加 VirtualPointer.lib

// 3. 直接使用API
VPTR_HANDLE handle = VPTR_CreateAdapter("vPointer");
```

### 3. 基本使用示例

```cpp
#include "VirtualPointerCAPI.h"
#include <iostream>

int main() {
    // 创建并连接到设备
    VPTR_HANDLE handle = VPTR_CreateAndConnect("192.168.1.100", 6533);
    if (!handle) {
        std::cerr << "连接失败" << std::endl;
        return 1;
    }

    // 显示指针
    VPTR_ShowPointer(handle);
    
    // 移动指针
    VPTR_MovePointer(handle, 100, 200);
    
    // 模拟按下
    VPTR_SetPointerPressed(handle, true);
    Sleep(200);
    VPTR_SetPointerPressed(handle, false);
    
    // 隐藏指针
    VPTR_HidePointer(handle);
    
    // 清理
    VPTR_DestroyAdapter(handle);
    return 0;
}
```

## API 参考

### C++ 接口

#### 核心类：IVirtualPointerAdapter

```cpp
namespace vptr {

class IVirtualPointerAdapter {
public:
    // 生命周期管理
    virtual bool Initialize(const AdapterConfig& config) = 0;
    virtual void Shutdown() = 0;
    virtual bool Connect() = 0;
    virtual void Disconnect() = 0;
    
    // 指针控制
    virtual bool MovePointer(int32_t x, int32_t y) = 0;
    virtual bool SetPointerPosition(const PointerPosition& position) = 0;
    virtual bool ShowPointer() = 0;
    virtual bool HidePointer() = 0;
    virtual bool SetPointerPressed(bool pressed) = 0;
    
    // 状态查询
    virtual PointerState GetPointerState() const = 0;
    virtual PointerPosition GetPointerPosition() const = 0;
    virtual ConnectionState GetConnectionState() const = 0;
    virtual DeviceInfo GetDeviceInfo() const = 0;
    virtual bool IsDeviceOnline() const = 0;
    
    // 回调设置
    virtual void SetOrientationCallback(OrientationCallback callback) = 0;
    virtual void SetConnectionCallback(ConnectionCallback callback) = 0;
    virtual void SetErrorCallback(ErrorCallback callback) = 0;
    
    // 其他
    virtual bool SendKeepAlive() = 0;
    virtual const char* GetAdapterName() const = 0;
};

// 工厂函数
IVirtualPointerAdapterPtr CreateVPointerAdapter();

} // namespace vptr
```

## 回调机制

### 方向变化回调

```cpp
// C++
adapter->SetOrientationCallback([](DeviceOrientation orientation) {
    switch (orientation) {
        case DeviceOrientation::Portrait: 
            std::cout << "竖屏" << std::endl; 
            break;
        case DeviceOrientation::LandscapeRight: 
            std::cout << "横屏右" << std::endl; 
            break;
        // ...
    }
});
```

```c
// C
void OnOrientationChanged(VPTR_ORIENTATION orientation, void* userData) {
    // 处理方向变化
}

VPTR_SetOrientationCallback(handle, OnOrientationChanged, NULL);
```

### 连接状态回调

```cpp
adapter->SetConnectionCallback([](ConnectionState state, const std::string& msg) {
    if (state == ConnectionState::Connected) {
        std::cout << "已连接: " << msg << std::endl;
    } else if (state == ConnectionState::Disconnected) {
        std::cout << "已断开: " << msg << std::endl;
    }
});
```

## 扩展开发

### 添加新的适配器

如需支持其他虚拟指针APP，只需实现`IVirtualPointerAdapter`接口：

```cpp
class MyAppAdapter : public IVirtualPointerAdapter {
public:
    bool Initialize(const AdapterConfig& config) override {
        // 实现初始化
    }
    
    bool MovePointer(int32_t x, int32_t y) override {
        // 实现移动逻辑
    }
    
    // ... 实现其他纯虚函数
};

// 注册到工厂
VPTR_C_API VPTR_HANDLE VPTR_CreateAdapter(const char* adapterType) {
    if (strcmp(adapterType, "MyApp") == 0) {
        return RegisterAdapter(std::make_shared<MyAppAdapter>());
    }
    // ...
}
```

## 常见问题

### Q1: 连接失败怎么办？

1. 检查设备和电脑是否在同一WiFi网络
2. 确认vPointer APP已启动并正在运行
3. 检查IP地址和端口是否正确
4. 关闭防火墙或添加例外规则

### Q2: 指针位置不准确？

vPointer使用绝对像素坐标。需要确保：
1. 坐标值在Android屏幕分辨率范围内
2. 考虑设备方向变化

### Q3: 如何获取设备方向？

设置方向回调函数，当设备旋转时会自动触发：

```cpp
adapter->SetOrientationCallback([](DeviceOrientation ori) {
    // ori: Portrait, LandscapeRight, PortraitUpsideDown, LandscapeLeft
});
```

### Q4: 支持多设备吗？

支持。创建多个适配器实例，每个对应一个设备：

```cpp
auto device1 = VPTR_CreateAndConnect("192.168.1.100", 6533);
auto device2 = VPTR_CreateAndConnect("192.168.1.101", 6533);
```
